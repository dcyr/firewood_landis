Description of contents 
*****************************

This folder contains all input files for LANDIS-ForCS simulations in the two target landscape. 

It also includes the documentation on parameters for ForCS input file (ForCS_params.xlsx and ForCS_supplMat.docx)

The folder 'biomass_succession_inputfiles' contains input files for biomass-successison prepared by Y.Boulanger. 
MaxANPP, MaxBiom and Pest tables were used as input in ForCS-succession.
