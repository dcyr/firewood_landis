Description of contents 
*****************************

This folder contains all input files for LANDIS-ForCS simulations in the two target landscapes: MRC Outauais (MRCOuta) and MRC Centre. 

The folder 'biomass_succession_inputfiles' contains input files for biomass-successison prepared by Y.Boulanger. 
MaxANPP, MaxBiom and Pest tables were used as input in ForCS-succession. 

Specific information on the parameterization of ForC_succession input files can be found in firewood_landis/documentation
